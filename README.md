# Basic Quiz App
 My first app made using simple widgets and has a basic UI.
The main file contains the code part for basic layout of the app.
The quiz file contains the code part for the question-answer combination .
The question and answer file as the name contains code part for questions and answer widget.
The result file contains the code for reseting quiz and for deciding the negativity of the person as per the option selected by the person for each question.
